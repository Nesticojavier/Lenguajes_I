Buddy.kt: archivo que implementa la clase Buddy que simula un Buddy System
Main.kt:  cliente para probar el simulador implementado en Buddy.kt e interactuar con el usuario

Para compilar:
>make

Para ejecutar:
>./runBuddy.sh $argument1
siendo este argumento la cantidad de memoria

No se usó ninguna herramienta para pruebas unitarias.
En lugar de eso se ejecuto el programa con una cantidad fija de memoria disponible
y se reservó  y liberó memoria para verificar el funcionamiento.
